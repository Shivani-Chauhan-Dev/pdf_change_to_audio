# PDF to Audio Converter

This is a simple PDF to Audio Converter script that converts PDF documents into audio files. It utilizes Python libraries to achieve this functionality.

## Features
1. Converts PDF files into audio files.
3. Easy-to-use command-line interface.

## Installation

1. Clone this repository:

    ```bash
    git clone https://github.com/Shivani-Chauhan-Dev/pdf_change_to_audio`

2. Navigate to the project directory:
 
    ```bash 
    cd pdf_change_to_audio

## Install the required dependencies:

    pip install PyPDF2

    pip install pyttsx3


4. Run the main.py file using Python 3.

## Requirements
- Python 3.x
- PyPDF2 library
- pyttsx3 library
